# 《缘合》产品路线图 — 工具先行策略

## 战略总纲

**核心策略：** 先做个人命理工具获取用户和数据，再叠加社交匹配功能。

**三阶段路径：**
```
Phase 1（工具）→ Phase 2（社区）→ Phase 3（匹配社交）
"了解自己"    → "找到同类"    → "遇见对的人"
```

**为什么不直接做社交：**
- 社交App的冷启动是致命问题——没有用户就没有匹配，没有匹配就留不住用户
- 工具产品的留存不依赖双边网络效应，单人也有价值
- 积累用户的八字数据后，匹配引擎的质量会远高于冷启动时的盲配
- Co-Star（星座工具→社交）、Keep（健身工具→社区）、LinkedIn（简历→社交）都验证了这条路径

## Phase 1：个人命理工具（第1-3个月）

### 目标
- 验证核心假设：用户愿意通过八字/命理了解自己
- 获取首批 **5万+** 注册用户
- 日活留存率 > **30%**（7日）

### 产品形态
**微信小程序 + 独立App同步开发**（小程序负责获客，App负责留存）

### 核心功能

#### 1.1 八字性格卡片（P0 — 核心入口）
- 用户输入出生年月日时 → 30秒生成个性化"性格卡片"
- 卡片设计：视觉精美、可直接分享到朋友圈/小红书
- 内容包括：核心性格特质（3-5个标签）、情感模式、隐藏优势、关系倾向
- **表达方式：** 现代心理学语言，不用传统命理术语
- 正例："你像一棵正在生长的树——内心有强烈的向上力量，但需要稳定的滋养"
- 反例：~~"日主甲木，偏弱，喜水木"~~

#### 1.2 每日状态卡（P0 — 核心留存）
- 每日早8点推送个性化运势/状态卡
- 内容基于八字+当日天干地支的交互计算
- 包含：今日情绪基调、适合做什么、人际提醒、一句话洞察
- **设计参考Co-Star：** 简洁、有态度、有洞察感，不是泛泛的"今天运势不错"

#### 1.3 关系分析卡（P0 — 裂变引擎）
- 输入两个人的生辰 → 生成双人关系分析
- 适用场景：和暧昧对象合不合、和闺蜜/兄弟的友谊模式、和父母的相处方式
- 这是**天然的社交裂变功能**——用户会主动把卡片发给对方
- 每人每天免费1次关系分析，更多次数需邀请好友或付费

#### 1.4 性格档案（P1）
- 用户的完整个人命理主页
- 包含：性格全景、情感模式、事业倾向、关系偏好、年度节奏
- 可逐步解锁更多维度（通过持续使用或付费）

### 不做的事情
- ❌ 用户匹配/推荐
- ❌ 聊天/IM
- ❌ 任务系统
- ❌ 公益功能
- ❌ 社区/动态流

### 技术要点
- 八字排盘：使用开源库（lunar-python/lunar-java），不自研
- AI架构：规则引擎（八字计算）→ 结论骨架 → LLM润色表达
- 用户数据：核心分析结论持久化存储，确保一致性
- 小程序+App共用后端API

### 增长策略
- **小红书/抖音内容营销：** 生成的性格卡片天然适合社交媒体传播
- **关系分析裂变：** "输入你和TA的生辰看看合不合"是最强裂变点
- **命理/塔罗社群精准获客：** 豆瓣、微信群、小红书命理兴趣社群
- **节日/节气运营：** 春节运势、情人节关系分析、生日月专属报告

### 变现（轻度）
- 基础性格卡：免费
- 详细分析报告（深度版）：9.9-29.9元/份
- 关系深度分析：19.9元/份
- 年度运势报告：39.9元/份

### 关键指标
| 指标 | 目标值 |
|------|--------|
| 累计注册用户 | 5万+ |
| 7日留存率 | >30% |
| 每日状态卡打开率 | >40% |
| 关系分析卡分享率 | >25% |
| 付费转化率 | >3% |
| 单用户获取成本（CAC） | <5元 |

## Phase 2：轻社区 + 单人互动深化（第4-6个月）

### 前置条件
- Phase 1 的7日留存率达到25%以上
- 累计注册用户达到3万以上

### 目标
- 用户开始在App内产生社交行为
- 建立"同类人"连接感
- DAU > **1万**

### 新增功能

#### 2.1 塔罗互动（P0）
- 每日一抽：免费的日常塔罗牌
- 专题抽牌：感情、事业、选择题（"我该不该..."）
- 互动式体验：用户选牌 → AI结合八字给出个性化解读
- 可分享的塔罗结果卡片

#### 2.2 AI命理问答（P0）
- 基于用户八字档案的对话式问答
- 用户可以问："我为什么总遇到渣男""我适合什么类型的伴侣""最近为什么总焦虑"
- AI回答必须引用用户的八字分析结论，保持一致性
- 每日免费3次，更多需付费

#### 2.3 "同命人"广场（P1）
- 按八字特征（如"日主同为甲木"）或性格标签自动分组
- 匿名讨论区：用户用性格标签作为身份（不露脸、不露名）
- 话题引导：系统每天发起讨论话题（"甲木人最受不了什么类型的人？"）
- **这是从工具到社交的过渡层**——用户开始习惯在App里和人互动

#### 2.4 性格周报/月报（P1）
- 基于用户本周/本月的八字运势+使用数据生成回顾
- 包含：本周情绪趋势、关系洞察、下周预览
- 高分享价值的内容载体

#### 2.5 梅花易数（P2）
- 轻量级"起卦"功能
- 适合即时决策场景："今天适合表白吗""这份工作该接吗"
- 结合八字给出个性化建议

### 变现（扩展）
- AI问答包月会员：19.9元/月
- 塔罗深度解读：9.9元/次
- 详细月度报告：12.9元/月
- 会员订阅（全功能）：39.9元/月

### 关键指标
| 指标 | 目标值 |
|------|--------|
| DAU | 1万+ |
| 30日留存率 | >20% |
| AI问答人均对话数 | >3轮/周 |
| 同命人广场参与率 | >15% DAU |
| 付费会员转化率 | >5% |
| 月ARPU（付费用户） | >25元 |

## Phase 3：社交匹配 + 任务系统（第7-12个月）

### 前置条件
- 累计注册用户达到 **20万+**
- DAU 稳定在 **2万+**
- 同命人广场证明用户有社交意愿

### 目标
- 完成从工具到社交平台的跃迁
- 验证"八字匹配+任务破冰"的核心机制

### 新增功能

#### 3.1 匹配推荐（P0）
- 基于积累的八字数据+行为数据+注册问卷进行匹配
- 每日推荐3-5人，附"为什么你们匹配"的命理解释
- 用户可设置匹配偏好（性别取向、年龄范围、地理位置）

**匹配权重（初版）：**
| 维度 | 权重 | 数据来源 |
|------|------|----------|
| 八字契合度 | 35% | 八字排盘计算 |
| 性格标签相似/互补 | 25% | 八字衍生标签 |
| 价值观匹配 | 25% | 注册问卷（5个核心问题） |
| 行为活跃度 | 15% | App使用数据 |

#### 3.2 破冰任务（P0）
- 匹配成功后，系统推送一个轻量级共同任务替代"你好在吗"
- 任务类型：
  - **问答接龙：** "用三个词形容你理想中的周末"→对方回答→系统生成对比分析
  - **选择题PK：** "猫还是狗？""海边还是山里？"→看答案重合度
  - **共创任务：** 一起给一张AI生成的图片起名字
  - **命理话题：** "你的八字说你是XX型人，你觉得准吗？"
- 任务完成后自动开启聊天

#### 3.3 聊天（P0）
- 基础IM功能：文字、语音、图片
- AI破冰建议：根据双方八字分析，推荐聊天话题
- "关系进展"提示：系统根据聊天频率和内容，适时推送双人深度任务

#### 3.4 双人任务（P1）
- 关系深化机制：
  - **36问挑战：** 改编自心理学经典的"36个让人坠入爱河的问题"，用命理角度重新设计
  - **共同探索：** 一起做一个塔罗牌解读、一起看两个人的合盘分析
  - **线下引导：** "你们都是火象属性，适合一起去户外——这周约一次徒步？"

#### 3.5 "We Met" 反馈环（P1）
- 参考Hinge：匹配后一段时间，询问"你们见面了吗？感受如何？"
- 用真实反馈优化匹配算法
- 成功案例（用户允许后）可作为社区内容

### 变现（完整）
| 项目 | 价格 | 说明 |
|------|------|------|
| 基础会员 | 29.9元/月 | AI问答无限、塔罗深度、详细报告 |
| 高级会员 | 59.9元/月 | 基础会员 + 每日10个匹配 + 高级筛选 |
| 单次匹配加速 | 6元/次 | 额外获得一个匹配推荐 |
| 双人深度合盘 | 29.9元/份 | 两人的详细关系分析报告 |
| 超级喜欢 | 3元/次 | 让对方优先看到你 |

### 关键指标
| 指标 | 目标值 |
|------|--------|
| 累计注册用户 | 50万+ |
| DAU | 5万+ |
| 匹配接受率 | >30% |
| 破冰任务完成率 | >50% |
| 匹配→聊天转化率 | >40% |

## 风险与应对

### 1. 监管风险（高）
- **风险：** 中国对"算命""占卜"有监管限制
- **应对：** 产品定位为"AI性格分析"/"心理洞察"工具，避免使用"算命""占卜""预测未来"等措辞。参考测测转型"AI泛心理"的策略。法律团队提前评估

### 2. Soul/测测快速跟进（中）
- **风险：** Soul加入八字功能或测测加强社交匹配
- **应对：** 速度是关键。Phase 1 用3个月快速上线，用"任务系统"建立不易复制的壁垒（需要深度的命理+心理学内容设计）

### 3. LLM成本失控（中）
- **风险：** 深度分析报告的LLM调用成本过高
- **应对：** 高频内容（每日推送）用小模型或预生成模板；深度分析（付费报告）用大模型。预估单用户月LLM成本控制在0.5元以内

### 4. 命理内容质量（中）
- **风险：** 八字分析不准确/前后矛盾，用户社群传播后信任崩塌
- **应对：** 核心结论持久化缓存；命理顾问定期审核AI输出；建立用户反馈机制

### 5. 工具→社交跃迁失败（中高）
- **风险：** 用户把产品当纯工具用，不愿意社交
- **应对：** Phase 2 的"同命人广场"是关键测试——如果匿名社区都没人参与，需要重新评估社交方向。此时产品仍可以作为纯工具盈利（付费报告）

## 决策检查点（Go/No-Go）

在每个Phase结束时做正式评估，决定是否进入下一阶段：

### Phase 1 → Phase 2 检查点（第3个月末）
| 条件 | Go | Pivot |
|------|-----|-------|
| 注册用户 | ≥3万 | <1万 → 重新评估获客策略 |
| 7日留存 | ≥25% | <15% → 产品价值不足，需重做 |
| 卡片分享率 | ≥20% | <10% → 裂变失败，需换增长方式 |
| 付费转化 | ≥2% | <0.5% → 付费价值不够 |

### Phase 2 → Phase 3 检查点（第6个月末）
| 条件 | Go | Pivot |
|------|-----|-------|
| DAU | ≥8000 | <3000 → 留存有问题 |
| 同命人广场参与率 | ≥10% DAU | <5% → 用户无社交意愿，转纯工具 |
| AI问答使用频率 | ≥2次/周/活跃用户 | <1次 → AI价值不足 |
| 付费会员 | ≥500人 | <100人 → 商业模式需调整 |

### Phase 3 验证检查点（第12个月末）
| 条件 | 成功 | 需调整 |
|------|------|--------|
| 匹配→聊天转化率 | ≥30% | <15% → 匹配质量差 |
| 破冰任务完成率 | ≥40% | <20% → 任务设计有问题 |
| 月营收 | ≥50万 | <10万 → 商业模式需重构 |
| NPS | ≥40 | <20 → 用户体验有根本问题 |
