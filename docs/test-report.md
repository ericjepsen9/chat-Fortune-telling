## 二、八字引擎详细分析

### 2.1 四柱排盘精度（14/14）

**分析方法**：7组出生时间，每组同时用引擎和lunar-javascript直接API双重输出对比。覆盖立春年柱分界、芒种月柱分界、子时跨日。

| 测试case | 引擎 | lunar-js | 验证重点 |
|----------|------|---------|---------|
| 1971-05-27 亥时 | 辛亥 癸巳 壬子 辛亥 | 同 | 月柱节气分界（原始bug case） |
| 2000-02-04 19时 | 己卯年 | 同 | 立春前=旧年（精确到小时） |
| 2000-02-04 21时 | 庚辰年 | 同 | 立春后=新年 |
| 2000-01-01 子时 | 己卯 丙子 戊午 壬子 | 同 | 跨年+子时 |
| 2024-06-04 午时 | 己巳月 | 同 | 芒种前=巳月 |
| 2024-06-06 午时 | 庚午月 | 同 | 芒种后=午月 |
| 1990-08-15 未时 | 庚午 甲申 壬子 丁未 | 同 | 常规验证 |

**结论**：引擎与lunar-javascript输出100%一致。

### 2.2 真太阳时校正（3/3）

- 经度校正：(87.6-120)*4 = -129.6分钟
- 均时差（8月中旬）= +2分钟
- 总校正 = -130分钟
- 14:00-130分 = 11:50 -> 未时退到午时
- 验证：时柱从丁未变为丙午

### 2.3 身强弱+格局（4/4）

**身强**：壬水日主，水5+金2.5，helpRatio>0.45 -> 身强

**身弱**：甲木日主，申月金旺克木，helpRatio<0.3 -> 身弱

**格局判定过程**：壬日主 -> 月支巳 -> 藏干丙庚戊 -> 看透出(均无) -> 取本气丙 -> 偏财格

**从格**：丁亥壬子丁亥庚子，火极弱水极旺 -> 从弱格

### 2.4 月令司令（2/2）

1990-08-15，立秋8月8日到白露9月8日共31天。8月15日在立秋后第7天，frac=7/31=23%。月支申藏庚壬戊。frac<0.3 -> 余气段 -> 戊土当令。

## 三、天文计算精度

### 3.1 太阳（1/2 引擎正确）

1990-08-15计算值142.16度，标准答案约142.2度，误差0.04度。

春分case计算值0.038度（极接近0度），引擎正确。测试代码范围"359-1"无法处理跨0度边界。

### 3.2 月亮（1/1）

8/15到8/16日移14.95度，符合真实月亮12-15度/天。

### 3.3 行星（2/2）

Mercury距日24.1度（合理范围28以内），Venus距日17.8度（合理范围47以内）。

三层计算：Keplerian -> 摄动修正(Jupiter-Saturn大不等式0.8度) -> 日心转地心。修复前Mercury偏太阳140度，修复后24度。

### 3.4 上升星座（1/1）

24小时覆盖10个不同星座。纬度敏感：北京3.3度/上海12.1度/广州8.6度。
