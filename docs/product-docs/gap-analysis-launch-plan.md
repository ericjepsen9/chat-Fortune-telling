### 第一个月目标
- 完成B1-B10全部后端基建
- 内部Alpha可用

### 核心风险
1. **AI安全**：命理+社交+AI三重敏感，必须在公开前完成安全控制
2. **匹配冷启动**：初期用户少，匹配体验差 → 需要种子用户策略
3. **八字精度**：专业用户会检验排盘准确性 → lunar-js升级是刚需
4. **合规**：中国区涉及"占卜"定性、用户出生信息属于敏感个人信息(PIPL) → 需法律咨询
