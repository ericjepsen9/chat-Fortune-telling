# 《缘合》产品功能结构文档（完整版）

## 一、产品核心逻辑

### 产品核心链路

自我认知 → 人格标签 → 匹配关系 → 任务互动 → 关系发展 → 长期陪伴

### 核心定位

通过AI与命理帮助用户理解自己，并建立更高质量的人际关系。

---

## 二、核心功能模块

### 1. 用户系统
- 注册/登录（手机号/微信）
- 出生信息录入
- 用户隐私与数据管理
- 用户标签初始化

### 2. 八字与人格系统
- 八字排盘（四柱、五行、十神）
- 人格标签生成（性格/关系/行为）
- 命局复杂度与安全等级
- 标签统一输出结构

### 3. 性格卡片系统
- 人格卡
- 关系模式卡
- 优势卡
- 风险提示卡
- 行动建议卡
- 分享卡片

### 4. 每日状态系统
- 今日关键词
- 今日关系建议
- 今日行动建议
- 推送提醒
- 打开记录

### 5. 关系分析系统
- 双人关系分析
- 匹配类型（互补/相似/成长）
- 优势与风险分析
- 相处建议
- 报告生成（付费）

### 6. AI问答系统
- 命局问答
- 关系建议
- 情绪分析
- 行动指导
- 问题分类与安全控制
- **支持四种占术模式：**
  - 八字命理（四柱八字·五行十神）
  - 梅花易数（即时起卦·决策分析）
  - 塔罗牌（牌阵解读·直觉指引）
  - 印度占星术（星盘解析·业力指引）

### 7. 塔罗与轻占卜
- 单张抽牌
- 三张牌阵
- 情绪/关系解读

### 8. 梅花易数
- 即时决策分析
- 当前状态判断
- 行动建议

## 十、总结

本产品不是算命工具，也不是传统社交产品。

核心是：
- 通过"理解自己"降低社交门槛
- 通过"结构化匹配"提升关系质量
- 通过"任务互动"推动关系发展

**最终目标：让关系更容易发生，并更容易走下去。**
