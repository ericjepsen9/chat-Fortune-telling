## 技术部署架构（亚洲+北美双区）

### 基础设施

```
                    ┌──── 全球 CDN (Cloudflare) ────┐
                    │                                │
          ┌────────┴────────┐              ┌────────┴────────┐
          │   亚洲区域        │              │   北美区域        │
          │   (阿里云 香港)   │              │   (AWS us-west-2) │
          ├─────────────────┤              ├──────────────────┤
          │ App Server (K8s) │              │ App Server (ECS)  │
          │ Redis Cluster    │              │ ElastiCache       │
          │ PostgreSQL (RDS) │              │ RDS PostgreSQL    │
          │ OSS (文件存储)    │              │ S3 (文件存储)     │
          ├─────────────────┤              ├──────────────────┤
          │ LLM网关:         │              │ LLM网关:          │
          │ MiniMax (主)     │              │ OpenAI GPT-4o (主) │
          │ DeepSeek (备)    │              │ Claude (备)        │
          └─────────────────┘              └──────────────────┘
                    │                                │
                    └──── 数据不跨区（合规）────────┘
```

### 区域路由策略

```
用户请求 → Cloudflare DNS (Geo)
  │
  ├── IP在亚洲 → 阿里云香港节点
  │   ├── 数据库：阿里云RDS（数据留亚洲）
  │   ├── LLM：MiniMax-M2.7 / DeepSeek
  │   ├── 合规：PIPL（中国个人信息保护法）
  │   └── 语言：默认中文
  │
  └── IP在北美/欧洲 → AWS us-west-2
      ├── 数据库：AWS RDS（数据留北美）
      ├── LLM：GPT-4o-mini / Claude Haiku
      ├── 合规：CCPA/CPRA/GDPR
      └── 语言：默认English，可切中文
```

├── 前端 i18n

|------|------|---------|
| S级（绝密） | 精确出生时间、完整八字四柱 | AES-256加密存储，客户端解密，不明文传输 |
| A级（高敏） | 日主/十神/格局、性格分析 | 加密存储，不对其他用户可见 |
| B级（中敏） | 匹配分数、AI占卜报告 | 本地缓存，服务端不持久化内容 |
| C级（低敏） | 昵称、星座、兴趣标签 | 正常存储，可对其他用户展示 |

### 对其他用户的信息隔离

```
默认可见：昵称、年龄段(25-30)、匹配等级(高/中/低)、兼容类型
需对方授权：精确年龄、星座、五行属性、匹配分数
永远不可见：出生日期、时辰、八字、AI报告、与他人匹配记录
```

### 3级隐私档位（用户自选）

```
标准模式（默认）：年龄段+匹配等级+兼容类型
开放模式：+ 精确年龄+星座+五行属性+匹配分数
保守模式：只显示昵称+大致距离
```

### 数据收集最小化

```
必须收集：出生日期（命理计算）、性别（匹配）、邮箱（验证）
可选收集：出生时辰、出生城市、昵称（需单独授权弹窗）
绝不收集：真实姓名、身份证号、精确GPS、通讯录、相册
```

### LLM调用隐私

```
发给AI：匿名出生数据（年月日时+性别）→ 无法关联到用户
不发给AI：真实姓名、手机号、位置、用户ID
AI结果：不持久化到服务端，只在客户端缓存
调用日志：只记录模式/耗时/token数，出生信息用****替代
```

### 用户权利（CCPA/CPRA）

```
知情权：查看所有已收集数据
可携带权：导出JSON/PDF
删除权：各项数据独立可删，注销72小时内完全清除
退出权：退出匿名分析/个性化推荐/第三方共享
更正权：随时修改出生信息
```
