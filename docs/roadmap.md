# 缘合 — 功能阶段规划

## 阶段总览

| 阶段 | 定位 | 核心目标 | 预估周期 | 状态 |
|------|------|---------|---------|------|
| P1 | 分析 + 陪伴产品 | 验证核心价值，让用户认识自己 | - | ✅ 原型完成 |
| P1.5 | 体验增强 | 补全关键体验，减少流失 | ~4周 | 📋 待开发 |
| P2 | 关系产品 | 从个人工具升级为双人关系平台 | ~6周 | 📋 待开发 |
| P3 | 商业化 + 生态 | 变现、公益、长期留存 | ~8周 | 📋 待开发 |

---

## P1 — 分析 + 陪伴产品 ✅

### 已完成功能

| 模块 | 功能 | 说明 |
|------|------|------|
| 用户系统 | 出生信息录入 | 年/月/日/时辰/性别 |
| 八字系统 | 八字排盘 + 人格标签 | 分析动画 + 标签生成 |
| 性格卡片 | 5张卡片 | 人格/关系模式/优势/风险/行动建议 |
| 每日状态 | 关键词 + 建议 | 今日关系建议、行动建议 |
| AI问答 | 四种占术模式 | 八字命理/梅花易数/塔罗牌/印度占星 |
| 好友私聊 | 一对一实时聊天 | 含AI话题建议、在线状态、未读数 |
| 关系匹配 | 每日推荐 | 匹配度/类型/推荐理由 |
| 社区 | 同类人社区 | 五行标签分组、帖子浏览 |
| 任务 | 行为任务 | 表达/情绪/关系练习 |
| 导航 | 5个底部Tab | 首页/消息/AI问答(中心)/发现/我的 |

---

## P1.5 — 体验增强（~4周）

> 目标：补全P1核心体验中用户最容易流失的缺口

### 功能清单

| 功能 | 优先级 | 说明 | 核心价值 |
|------|--------|------|---------|
| 双人关系分析 | P0 | 输入双方出生信息，生成匹配报告 | 从"认识自己"到"理解关系"的桥梁 |
| 破冰任务系统 | P0 | 匹配后自动下发话题/问答/合作任务 | 解决"匹配了不知道聊什么" |
| 推送提醒系统 | P0 | 每日状态/匹配/任务/报告通知 | 没有推送，日活无法维持 |
| 分享卡片 | P1 | 性格卡片和分析报告生成分享图片 | 自然社交传播 |
| 情绪陪伴记录 | P1 | 心情打卡、情绪趋势、AI安抚 | 增强日常打开理由 |
| AI安全控制 | P0 | 问题分类、敏感内容拦截、安全等级 | AI上线后必须补全 |

### 开发建议

- 第1-2周：双人关系分析 + AI安全控制
- 第3周：破冰任务 + 推送提醒
- 第4周：分享卡片 + 情绪陪伴

---

## P2 — 关系产品（~6周）

> 目标：从个人分析工具升级为双人关系深度互动平台

### 功能清单

| 功能 | 优先级 | 说明 | 核心价值 |
|------|--------|------|---------|
| 双人任务系统 | P0 | 双人目标、任务打卡、共同记录 | 让关系有"推进感" |
| 关系阶段系统 | P0 | 初识→熟悉→升温→稳定→成长 | 每阶段解锁不同互动 |
| AI关系教练 | P0 | 聊天建议、关系判断、情绪引导 | 从被动回答变主动教练 |
| 默契值/进度条 | P1 | 双人互动量化指标 | 可视化关系发展 |
| 关系反馈系统 | P1 | 匹配质量反馈、阶段反馈 | 优化匹配算法 |
| 周报/月报 | P1 | 状态总结、关系回顾、成长记录 | 高级版试水付费 |

### 开发建议

- 第1-2周：关系阶段系统 + 双人任务
- 第3-4周：AI关系教练
- 第5周：默契值/进度条 + 关系反馈
- 第6周：周报/月报

---

## P3 — 商业化 + 生态（~8周）

> 目标：实现变现，构建公益差异化，建立长期留存

### 功能清单

| 功能 | 优先级 | 说明 | 核心价值 |
|------|--------|------|---------|
| 付费系统 | P0 | 深度报告/高级分析/AI次数 | 核心变现 |
| 会员系统 | P0 | 匹配增强/高级筛选/AI深度/专属内容 | 订阅收入 |
| 双人成长系统 | P1 | 双人周报、里程碑、成长记录 | 长期留存 |
| 公益任务系统 | P1 | 单人/双人公益、成就、纪念卡 | 品牌差异化 |
| 高级筛选 | P1 | 按五行/性格/关系类型精准筛选 | 会员专属价值 |
| 纪念卡生成 | P2 | 关系里程碑自动生成纪念卡 | 情感价值 + 传播 |
| 内容审核 | P0 | 社区内容审核系统 | 合规必须 |
| 数据加密 | P0 | 全链路数据加密、隐私保护 | 安全合规 |

### 开发建议

- 第1-2周：付费系统 + 会员系统
- 第3-4周：数据加密 + 内容审核
- 第5-6周：双人成长 + 公益任务
- 第7-8周：高级筛选 + 纪念卡

---

## 贯穿全程的技术建设

| 模块 | 说明 | 持续优化 |
|------|------|---------|
| 三层AI架构 | 规则层→骨架层→LLM层 | 每阶段根据反馈调权重 |
| 数据系统 | 用户画像、八字档案、标签、缓存 | 从轻量缓存升级到完整系统 |
| 推送系统 | P1.5引入，后续持续完善 | 智能推送策略优化 |
| 安全合规 | P1.5引入基础，P3全面加固 | GDPR等合规持续跟进 |

## 关键指标

| 阶段 | 北极星指标 | 辅助指标 |
|------|-----------|---------|
| P1 | 完成八字分析的用户比例 | 卡片浏览率、AI问答次数 |
| P1.5 | 次日留存率 | 推送打开率、破冰任务完成率 |
| P2 | 双人互动率 | 任务完成率、关系阶段升级率 |
| P3 | 付费转化率 | ARPU、会员续费率、公益参与率 |
