# 缘合 (YuanHe)

> 理解自己 · 遇见对的人

## 产品定位

通过AI与命理帮助用户理解自己，并建立更高质量的人际关系。

**核心链路：** 自我认知 → 人格标签 → 匹配关系 → 任务互动 → 关系发展 → 长期陪伴

## 产品阶段

| 阶段 | 定位 | 核心功能 | 预估周期 |
|------|------|---------|---------|
| **P1** | 分析 + 陪伴产品 | 八字分析、性格卡片、AI问答(4术)、好友私聊、社区 | ✅ 已完成原型 |
| **P1.5** | 体验增强 | 双人关系分析、破冰任务、推送提醒、情绪陪伴 | ~4周 |
| **P2** | 关系产品 | 双人任务、关系阶段、AI教练、默契值 | ~6周 |
| **P3** | 商业化 + 生态 | 付费报告、会员系统、公益任务、安全合规 | ~8周 |

## P1 原型说明

原型文件为 React JSX 组件，可在支持 React 的环境中直接渲染。

### 包含页面

- **启动页 & 引导页** — 品牌展示 + 3步产品介绍
- **出生信息录入** — 年/月/日/时辰/性别
- **八字分析动画** — 排盘进度展示
- **首页** — 今日状态、快捷入口、推荐用户、任务进度
- **AI问答** — 八字命理 / 梅花易数 / 塔罗牌 / 印度占星 四模式切换
- **消息列表 & 好友私聊** — 含在线状态、未读数、AI话题建议
- **关系匹配** — 匹配度、类型、推荐理由
- **同类人社区** — 五行标签分组、帖子互动
- **性格卡片** — 人格卡/关系模式卡/优势卡/风险卡/行动建议卡
- **每日状态** — 关键词、关系建议、行动建议
- **任务系统** — 表达/情绪/关系练习
- **个人中心** — 数据概览、功能入口

### 底部Tab栏（5个）

1. 首页
2. 消息（含未读红点）
3. AI问答（中间凸起金色按钮）
4. 发现（社区）
5. 我的

## 设计风格

- 深色东方神秘美学
- 主色调：金色 `#C9A96E`
- 五行色彩体系：金`#C9A96E` 木`#6BAF7B` 水`#5B8FB9` 火`#D4654A` 土`#B8935A`
- 字体：Noto Serif SC (标题) + Noto Sans SC (正文)

## 技术架构（规划）

- **三层AI架构**：规则层（八字/标签）→ 骨架层（结构输出）→ LLM层（自然语言）
- **实时通信**：WebSocket（好友私聊）
- **数据存储**：用户数据、八字档案、标签系统、分析缓存、聊天记录
